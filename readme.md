### Development setup

Make sure you have latest node and npm installed. 

Install MongoDB and run it. 

Install dependencies for both server and client parts of the application. 

`npm install`
`cd client`
`npm install` 
`cd ..`

Run both client and server: 

`npm run dev`

This should work.
